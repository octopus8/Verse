\doxysubsection{Detailed Description}
Used only for deployed Web\+GL apps, this component enables microphone support upon initial switch to VR, then destroys itself. 


\begin{DoxyItemize}
\item This component is added by O8\+CSystem on Awake if the app is deployed as a Web\+GL app.
\item On Web\+GL, if the microphone support is enabled on on app start and microphone permissions have not been granted yet, the browser will request microphone permissions. To allow the webpage to request permissions and respond to the user\textquotesingle{}s selection, microphone permissions are not enabled until the initial switch to VR, which happens after the microphone permission request in the app flow. 
\end{DoxyItemize}
