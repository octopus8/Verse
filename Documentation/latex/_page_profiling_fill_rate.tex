\hypertarget{_page_profiling_fill_rate_SectionProFillDesc}{}\doxysection{Scene Description}\label{_page_profiling_fill_rate_SectionProFillDesc}
The scene contains the {\ttfamily Platform Rig} and a single static lightmapped \& textured quad that sits in front of the camera such that when looking straight forward it takes up the full view. If the user looks up a certain amount, the view can be changed such that the quad takes up only the bottom half of the view. Looking away from the quad results in an empty view.
