\doxysubsection{Detailed Description}
Enables microphone support upon initial switch to VR. 

On Web\+GL, if the microphone support is enabled on on app start and microphone permissions have not been granted yet, the browser will request microphone permissions. To allow the webpage to request permissions and respond to the user\textquotesingle{}s selection, microphone permissions are not enabled until the initial switch to VR, which happens after the microphone permission request in the app flow.
